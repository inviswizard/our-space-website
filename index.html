Welcome! Let's set up your GitHub storage.

To use Our Space with GitHub file storage, you need to configure a few things first:

  1. Create a GitHub repository for your data
  2. Generate a Personal Access Token
  3. Configure the settings below

Step 1: Create GitHub Repository

1. Go to GitHub and create a new repository

2. Make it public or private (your choice)

3. Initialize with a README

4. Create a file called rooms_data_1.txt in the root

Step 2: Generate Access Token

1. Go to GitHub Tokens

2. Click "Generate new token" ‚Üí "Generate new token (classic)"

3. Give it a name like "Our Space App"

4. Select these scopes:

  • ‚úì repo (Full control of private repositories)
  • ‚úì public_repo (Access public repositories)

5. Click "Generate token" and copy it
